# IQ Banking Core

Spring Boot banking application for account management and transactions.

## Tech Stack
- Java 25
- Spring Boot 4.x
- Spring Data JPA / Hibernate
- MySQL 8.0
- Maven
- Git

## Features Implemented
- ✅ REST API for system health checks
- 🚧 Account CRUD operations (In Progress)
- 🚧 Transaction management (Planned)
- 🚧 JWT Security (Planned)

## API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/health | System health check |

git clone https://github.com/arislan404/banking-core.git
